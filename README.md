# modeline.github.io

The repository for the official landing page of the Modeline project. The `main` branch contains project codebase, while the generated static files are stored on `gh-pages`. The website is publicly accesible at https://modeline.github.io/.

## Getting Started

First, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
